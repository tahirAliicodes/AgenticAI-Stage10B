## 🚀 What This Project Does

Standalone **AI security components** — modular, plug-and-play security modules for any AI pipeline.

---

## ✅ Modules

- 🛡️ **Injection_detector.py** — prompt injection detection
- 🔏 **Pii_redactor.py** — PII removal from AI outputs  
- ✅ **ouput_filter.py** — response validation
- 📋 **compliance.py** — policy compliance checker
